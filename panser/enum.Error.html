The error type for panser-releated operations and associated traits.

Errors mostly originate from the dependencies, but custom instances of Error can be crated with the Generic variant and a message.

pub fn code(&self) -> i32

Gets an error code related to the error.

This is useful as a return, or exit, code for a command line application, where a non-zero integer indicates a failure in the application. It can also be used for quickly and easily teseting equality between two errors.
